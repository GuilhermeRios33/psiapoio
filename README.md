# 🧠 Psi Apoio

> Um aplicativo móvel voltado para o apoio psicológico e bem-estar emocional, oferecendo uma interface acolhedora e acessível.

O **Psi Apoio** foi desenvolvido para facilitar o acesso a recursos de saúde mental. Com uma interface ágil e fluida criada em Flutter e um backend seguro estruturado no Firebase, o aplicativo busca proporcionar um ambiente confiável para o suporte emocional e bem-estar do usuário.

## 🚀 Principais Funcionalidades

*   **Autenticação Segura:** Login rápido e gerenciamento de usuários utilizando o Firebase Authentication.
*   **Interface Fluida:** Design responsivo, amigável e focado na experiência do usuário, construído com os widgets do Flutter.
*   **Armazenamento em Nuvem:** Gerenciamento e sincronização de dados conectados diretamente com o banco de dados do Firebase.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído com as seguintes tecnologias e ferramentas:

*   **[Flutter](https://flutter.dev/):** Framework para o desenvolvimento da interface mobile (cross-platform).
*   **[Dart](https://dart.dev/):** Linguagem de programação base utilizada pelo Flutter.
*   **[Firebase](https://firebase.google.com/):** Backend as a Service (BaaS) utilizado para banco de dados e autenticação.
*   **[Android Studio](https://developer.android.com/studio):** Ambiente de Desenvolvimento Integrado (IDE) principal utilizado para desenvolvimento, emulação e testes do aplicativo.
